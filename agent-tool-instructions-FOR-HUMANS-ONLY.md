## Human-Only Features
Some of the features are for humans only, and are not intended to be used by AI agents. These features are not mentioned in the `--help` output of the command line tool.

## cite2md
- `--vs` : open in vs code (same as `code "$(cite2md [key])"`)
- `--vsi` : open in vs code insiders (same as `code-insiders "$(cite2md [key])"`)
- `-r` / `--reveal` : open Finder with the file revealed (same as `open -R "$(cite2md [key])"`)
